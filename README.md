El proyecto aborda el reto de identificar materiales en movimiento dentro de un entorno de computación de borde (edge computing), 
utilizando hardware de bajos recursos como la Raspberry Pi 3.

En la fase inicial se evalúa la implementación de un sistema de visión artificial aplicado a la clasificación automatizada 
de residuos sólidos (botellas PET, envases rígidos, empaques flexibles y Tetra Pak) para las estaciones de reciclaje Ecobot.
